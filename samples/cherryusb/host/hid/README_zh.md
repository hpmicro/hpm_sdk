# USB HID HOST

## 概述

本示例工程展示USB HID HOST

## 硬件设置

- 使用USB Type-C线缆连接PC USB端口和PWR DEBUG端口
- 使用USB Type-C线缆连接PC USB端口和开发板USB0端口

## 运行现象

- 将程序下载至开发板运行，使用串口调试助手查看输出log

### 接键盘

- 开发板USB0端口接入键盘时，串口调试助手会显示如下数据：

[D/USB] Port 1, status:0x101, change:0x00

[D/USB] Port 1, status:0x101, change:0x00
