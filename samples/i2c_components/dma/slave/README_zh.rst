I2C_COMPONENT_SLAVE_DMA
==============================================

概述
------

- i2c_components_master_dma示例工程展示了I2C作为从机，通过DMA传输数据的功能。

- 示例工程中的具体从机设备的寄存器地址的内存访问，I2C slave先读取寄存器地址数据，然后再进行数据的读取和写入

- 示例工程中的纯数据访问，I2C master先读取数据，之后再写入数据，并对比读取和写入的数据是否一致。

提示
------

- I2C component使用的DMA管理器组件，需要调用dma_mgr_init API初始化DMA管理器。

- 组件详情文档请参见 :ref:`i2c_components <i2c_component>`

硬件设置
------------

将两个板子的 :ref:`I2C引脚 <board_resource>` 相连。

运行现象
------------

- 先运行slave，再运行master
