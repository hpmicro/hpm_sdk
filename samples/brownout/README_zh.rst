.. _brownout:

Brownout中断
======================

概述
------

Brownout中断示例工程展示了VPMC欠压时触发brownout中断的功能。在brownout中断服务函数中，相应的GPIO引脚会拉高，通过逻辑分析仪或示波器可以观察到brownout中断的触发情况。

硬件设置
------------

参考开发板readme中的BROWNOUT中断指示引脚部分，连接逻辑分析仪或示波器。

运行现象
------------

当工程正确运行后，断开开发板电源，可以在逻辑分析仪或示波器上观察到brownout中断指示引脚拉高的波形，表示brownout中断被触发。
