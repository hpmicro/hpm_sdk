# h264 decoder
## 概述

解码例程实现了软件解码从uart接收的h264码流。本工程中，MCU会读取uart13中的码流数据，对其进行解码，输出图像到LCD上。

## 硬件设置

1. 连接开发板和LCD显示屏
2. 连接uart13到数据源串口


## 运行现象
