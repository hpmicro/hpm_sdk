# h264 decoder
## 概述

h264解码例程实现了软件解码h264压缩的mp4文件。本工程中，MCU会读取SD卡中的1.mp4文件，对其进行解码，输出图像到LCD上，同时测量解码每帧消耗的时间，并输出到串口中。

## 硬件设置

1. 连接开发板和LCD显示屏
2. 插入SD卡
