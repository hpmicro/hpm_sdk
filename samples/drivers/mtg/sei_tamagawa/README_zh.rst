MTG SEI tamagawa 例程
======================================

概述
------

MTG SEI tamagawa 例程演示了使用SEI与tamagawa编码器通信获取数据，经过MTG的滤波器进行处理，以及经过轨迹预测模块对其进行时间补偿的功能。例程输出SEI的原始数据和经过MTG处理后的数据，并输出两者之间的差值。

硬件设置
------------

- 多摩川编码器的型号为TS5700N8401。

- 将SEI接口信号DATA_P/DATA_N与编码器的数据信号相连接，编码器通过开发板+5V供电。
