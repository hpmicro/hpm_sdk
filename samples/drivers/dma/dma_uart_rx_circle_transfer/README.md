# DMA circularly receives uart data example、

## Overview

- This example project shows the DMA circularly transfer mode realized by chain transfer.

- The ChnLLPointer of the two chain descriptors point to each other, A point to B, and B point to A. After A transfer completed, B will be transfered, and after B transfer completed, A will be transfered, thus realizing circularly transfer without software intervention.

## Board Setting

- Use the serial port debugging assistant to send and receive data
- The serial port is set to ``115200bps``, ``1 stop bit``, ``no parity bit``
