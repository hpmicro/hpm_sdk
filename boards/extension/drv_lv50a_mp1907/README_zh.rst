DRV-LV50A-MP1907电机驱动板
=============================

- 使用跳线端子连接J12、J7、J6到HW、HV、HW
- 使用跳线端子连接J10的VCC到3V3
- 将电机的U、V、W三相和J4端子按照正确的线序相连
- 将电机的信号线插入J14插座
- 将24V电源接入J3端子

.. note::

    1. **请注意电源正负极方向**
    2. **插接电路板时不要带电操作**
    3. **单独给电机驱动板上电，观察电流是否异常。如果异常请检查线路连接**
    4. **进行下一步前，请确保已经给电机驱动板断电**
